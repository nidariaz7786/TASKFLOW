#  TaskFlow - Smart To-Do List Web Application

TaskFlow is a modern, responsive, and feature-rich To-Do List Web Application built with **HTML5, CSS3, Bootstrap 5, and Vanilla JavaScript**. It helps users organize tasks, track progress, and improve productivity with a clean and intuitive interface.

## 🌐 Live Demo

**🔗 Live Website:** https://YOUR-NETLIFY-LINK.netlify.app

##  GitHub Repository

**🔗 Repository:** https://github.com/nidariaz7786/TASKFLOW

---

##  Features

*  Create, edit, and delete tasks
*  Mark tasks as completed or pending
*  Dashboard with task statistics
*  Progress bar and completion percentage
*  Search, filter, and sort tasks
*  Due date support
*  Task categories and priorities
*  Dark Mode
*  Theme color customization
*  Profile page with productivity statistics
*  Export tasks as JSON
*  Local Storage support
*  Fully responsive design

##  Getting Started

### Clone the Repository

```bash
git clone https://github.com/nidariaz7786/TASKFLOW.git
```

### Navigate to the Project

```bash
cd TASKFLOW
```

### Run the Project

Open **index.html** in your browser, or use the **Live Server** extension in Visual Studio Code for a better development experience.
